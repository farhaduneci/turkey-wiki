# Turkish Date, Time & Number Reference

A quick-lookup reference for the words and formats you'll run into constantly on official documents, appointments, and daily life. See [Date and Number Formats in Turkey](../mistakes/date-number-formats.md) for why the `DD.MM.YYYY` format and separators matter and how getting them wrong caused real trouble.

## Days of the Week (Günler)

| Turkish | English |
|---|---|
| Pazartesi | Monday |
| Salı | Tuesday |
| Çarşamba | Wednesday |
| Perşembe | Thursday |
| Cuma | Friday |
| Cumartesi | Saturday |
| Pazar | Sunday |

Note: the week is generally considered to start on **Monday** in Turkey, and "Pazar" (Sunday) literally also means "market" / "bazaar" — don't confuse it with "Pazartesi" (Monday).

## Numbers (Sayılar)

| Turkish | Number |
|---|---|
| bir | 1 |
| iki | 2 |
| üç | 3 |
| dört | 4 |
| beş | 5 |
| altı | 6 |
| yedi | 7 |
| sekiz | 8 |
| dokuz | 9 |
| on | 10 |
| yirmi | 20 |
| otuz | 30 |
| kırk | 40 |
| elli | 50 |
| yüz | 100 |
| bin | 1,000 |
| milyon | 1,000,000 |

Turkish numbers compose the way English ones do (e.g. `yirmi bir` = 21, `yüz elli` = 150), but remember the [decimal/thousands separators are swapped](../mistakes/date-number-formats.md) compared to what you may be used to.

I learned these using [Turkish Numbers](https://games.farhaduneci.com/html/turkish-numbers), a small practice game — worth a try if you're still building number fluency.

## Telling Time

- "What time is it?" → **Saat kaç?**
- "It's 3 o'clock" → **Saat üç**
- Turkey uses the 24-hour clock on official schedules (transit, TV, appointments), e.g. `14:30`, even though people often say times the 12-hour way in conversation.

## Ordinal Numbers (for dates like "the 5th")

Add **-inci/-ıncı/-uncu/-üncü** (vowel harmony applies) to the number, e.g.:

| Turkish | English |
|---|---|
| birinci | 1st |
| ikinci | 2nd |
| üçüncü | 3rd |
| beşinci | 5th |
| onuncu | 10th |
