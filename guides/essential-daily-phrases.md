# Essential Daily Phrases

A minimal survival phrasebook for daily life in Turkey — greetings, shopping, directions, and what to say if something goes wrong.

## Politeness Level: "Sen" vs. "Siz"

Turkish has two forms of "you": informal **sen** and formal/plural **siz**. Use **siz** by default with strangers, officials, shopkeepers, and anyone older than you — switching to **sen** too early can come across as disrespectful. Wait until the other person uses **sen** with you, or until you know each other well.

## Greetings & Basics

| Turkish | English |
|---|---|
| Merhaba | Hello |
| Günaydın | Good morning |
| İyi günler | Good day |
| İyi akşamlar | Good evening |
| İyi geceler | Good night |
| Nasılsınız? | How are you? (formal) |

| Evet / Hayır | Yes / No |
| Lütfen | Please |
| Teşekkür ederim / Teşekkürler | Thank you |
| Rica ederim | You're welcome |
| Affedersiniz / Pardon | Excuse me / Sorry |
| Güle güle | Goodbye (said to the person leaving) |
| Hoşça kalın | Goodbye (said by the person leaving) |

## Getting By

| Turkish | English |
|---|---|
| Anlamıyorum | I don't understand |
| İngilizce biliyor musunuz? | Do you speak English? |
| Türkçe bilmiyorum | I don't speak Turkish |
| Yavaş konuşabilir misiniz? | Can you speak slowly? |
| Tekrar söyler misiniz? | Can you say that again? |
| ... nerede? | Where is ...? |
| Bu ne kadar? / Kaç para? | How much is this? |
| Çok pahalı | Too expensive |
| Hesap, lütfen | The bill, please |
| Yardım eder misiniz? | Can you help me? |

## Emergencies

| Turkish | English |
|---|---|
| İmdat! | Help! (urgent, e.g. danger) |
| Yardım edin! | Help me! |
| Polis | Police |
| Hastane | Hospital |
| Ambulans çağırın | Call an ambulance |
| Kayboldum | I'm lost |
| Pasaportumu kaybettim | I lost my passport |

**112** is the general emergency number in Turkey (police, ambulance, fire — all routed through one number).

## What Else Helps

- Numbers and telling time — see the [Date, Time & Number Reference](turkish-date-time-number-reference.md).
- Learning to read common signage: **Giriş** (Entrance), **Çıkış** (Exit), **Açık** (Open), **Kapalı** (Closed), **Çekiniz/İtiniz** (Pull/Push), **Kadın/Erkek** (Women/Men, for restrooms).
- Directions: **sağ** (right), **sol** (left), **düz** (straight), **yakın** (near), **uzak** (far).
